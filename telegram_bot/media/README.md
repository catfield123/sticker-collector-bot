# Instruction Video

Поместите видео с инструкцией в этот каталог с названием `instruction_video.mp4`.

Видео будет отправляться пользователям при выполнении команды `/start`.

## Требования:

- Название файла: `instruction_video.mp4`
- Формат: MP4 (поддерживается Telegram)
- Рекомендуемый размер: до 50MB для быстрой отправки

## Пример:

```bash
# Скопируйте ваше видео в эту директорию
cp /path/to/your/video.mp4 telegram_bot/media/instruction_video.mp4
```

Если видео отсутствует, бот будет работать нормально, но отправит текстовое сообщение вместо видео.
